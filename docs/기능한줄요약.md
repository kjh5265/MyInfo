# 기능 한줄 요약

## 전체 구조
- **React + Vite + Tailwind** 기반 개인 포트폴리오 사이트 (Vercel 배포)
- **Firebase Firestore** 실시간 DB, **Telegram Bot** 새 메시지 알림

## 기능 목록

| 기능 | 설명 |
|------|------|
| 프로필 카드 | 나이/지역/MBTI 등 기본 정보 표시 |
| 프로필 사진 | 클릭 3회 시 사진 교체 이스터에그 |
| 관심사 그리드 | 8가지 관심사 아이콘+텍스트 카드 |
| 음식 취향 | 좋아요/싫어요 카테고리 확장형 카드 |
| 다크모드 | OS 감지 + 수동 토글, localStorage 저장 |
| 소셜 링크 | 이메일/인스타그램 바로가기 버튼 |
| 1:1 채팅 | 사용자별 독립 대화방, 관리자가 개별 응답 가능 |
| 닉네임 설정 | 최초 채팅 시 닉네임 입력 (최대 10자) |
| 채팅 말풍선 | 카카오톡 스타일 날짜 구분선 + 시간 표시 |
| 관리자 로그인 | 비밀번호 인증으로 관리자 모드 진입 |
| 관리자 대화목록 | 모든 사용자 대화 목록 확인 및 선택 진입 |
| 채팅 초기화 | 관리자가 특정 대화를 소프트 삭제 (disabled 플래그) |
| 메시지 제한 | 최대 20자, 5분 내 20개, 하루 100개 제한 |
| 텔레그램 알림 | 사용자 메시지 수신 시 관리자에게 즉시 알림 |

## Firestore 구조 (1:1 채팅)

```
conversations/{userId}          # 대화 목록 메타데이터
  - authorName, lastMessage, lastMessageAt, userId

chats/{userId}/messages/{msgId} # 개별 대화 메시지
  - content, userId, authorName, isAdmin, disabled, createdAt
```
